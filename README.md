## 🚀 Overview

**AI Fitness Tracker** is a full-stack, AI-driven web platform that redefines nutrition tracking through **computer vision and intelligent automation**.

Upload a meal image → get **instant food recognition & calorie estimation** powered by Google Gemini.

Built with a **scalable architecture and premium UI**, delivering a seamless fitness tracking experience.

---

## ✨ Features

* 📸 **AI Food Snap** — Detect food & estimate calories via Gemini
* 📊 **Analytics Dashboard** — Track intake, burn & trends visually
* 🏃‍♂️ **Workout Logging** — Monitor activities & performance
* 🔐 **JWT Authentication** — Secure login & protected routes
* 🌙 **Modern UI/UX** — Glassmorphism + smooth animations

---

## 🧠 Architecture

```text
React (Client)
   ↓
Axios (JWT Layer)
   ↓
Strapi Backend
   ↓
Google Gemini AI
```

## ⚙️ Setup

```bash
git clone https://github.com/Ujjwal15-coder/AI-FITNESS-TRACKER.git
```

### Backend

```bash
cd server
npm install
npm run develop
```

### Frontend

```bash
cd client
npm install
npm run dev
```

---

## 🔑 Environment Variables

**Server**

```
GEMINI_API_KEY=your_api_key
```

**Client**

```
VITE_STRAPI_API_URL=http://localhost:1337
```
